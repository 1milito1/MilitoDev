Actualizado: 20/05/2025

Introducción al E-commerce con Next.js

Crear una tienda online moderna y escalable es más fácil que nunca con las herramientas adecuadas. En este tutorial, te guiaremos paso a paso para construir un e-commerce completo utilizando la potencia de Next.js para el frontend, Stripe para la gestión de pagos y Tailwind CSS para un diseño responsivo y personalizable. Aprenderás a:

  • Configurar un proyecto Next.js desde cero
  • Diseñar la interfaz de usuario con Tailwind CSS
  • Integrar la API de Stripe para procesar pagos
  • Gestionar productos y carritos de compra

Requisitos previos: Conocimientos básicos de JavaScript, React y Node.js. Asegúrate de tener Node.js y npm (o yarn) instalados en tu sistema.

Paso 1: Configuración del Proyecto Next.js

1

Crear un nuevo proyecto Next.js

Abre tu terminal y ejecuta el siguiente comando para crear un nuevo proyecto Next.js:

npx create-next-app my-ecommerce-app --typescript --tailwind --eslint

Este comando creará un proyecto Next.js con TypeScript, Tailwind CSS y ESLint ya configurados, lo cual es ideal para nuestro e-commerce.

2

Navegar al directorio del proyecto

cd my-ecommerce-app
3

Ejecutar la aplicación de desarrollo

Para verificar que todo está funcionando correctamente, inicia el servidor de desarrollo:

npm run dev

Abre tu navegador y ve a `http://localhost:3000` para ver la página de bienvenida de Next.js.

Paso 3: Integración de Stripe para Pagos

1

Instalar las dependencias de Stripe

Necesitamos instalar las librerías de Stripe para el frontend y el backend:

npm install @stripe/react-stripe-js @stripe/stripe-js stripe

`@stripe/react-stripe-js` y `@stripe/stripe-js` son para el frontend, y `stripe` es para el backend (API Routes de Next.js).

2

Configurar las claves de Stripe

Crea un archivo `.env.local` en la raíz de tu proyecto y añade tus claves de API de Stripe. Puedes obtenerlas desde tu dashboard de Stripe (usa las claves de prueba por ahora):

NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=pk_test_TU_CLAVE_PUBLICABLE
STRIPE_SECRET_KEY=sk_test_TU_CLAVE_SECRETA

Recuerda que las variables que empiezan con `NEXT_PUBLIC_` son accesibles en el lado del cliente.

3

Crear una API Route para el checkout

Crea un archivo `src/app/api/checkout_sessions/route.ts` (App Router) o `pages/api/checkout_sessions.ts` (Pages Router) para manejar la creación de sesiones de checkout de Stripe:

// src/app/api/checkout_sessions/route.ts (Ejemplo con App Router)
import { NextRequest, NextResponse } from 'next/server';
import Stripe from 'stripe';

const stripe = new Stripe(process.env.STRIPE_SECRET_KEY!, {
  apiVersion: '2023-10-16', // Usa la última versión de la API
});

export async function POST(req: NextRequest) {
  try {
    const { items } = await req.json();

    const lineItems = items.map((item: any) => ({
      price_data: {
        currency: 'usd',
        product_data: {
          name: item.name,
        },
        unit_amount: item.price * 100, // Stripe expects amount in cents
      },
      quantity: item.quantity,
    }));

    const session = await stripe.checkout.sessions.create({
      payment_method_types: ['card'],
      line_items: lineItems,
      mode: 'payment',
      success_url: `${req.nextUrl.origin}/success?session_id={CHECKOUT_SESSION_ID}`,
      cancel_url: `${req.nextUrl.origin}/cancel`,
    });

    return NextResponse.json({ sessionId: session.id });
  } catch (err: any) {
    return NextResponse.json({ statusCode: 500, message: err.message }, { status: 500 });
  }
}

Paso 4: Gestionar el Carrito de Compras

1

Implementar un Contexto de Carrito

Para gestionar el estado del carrito de compras de manera global, utilizaremos el Context API de React. Crea un archivo `src/context/CartContext.tsx`:

// src/context/CartContext.tsx
'use client'; // Necesario para componentes de cliente en App Router

import React, { createContext, useState, useContext, ReactNode } from 'react';

interface CartItem {
  id: number;
  name: string;
  price: number;
  quantity: number;
}

interface CartContextType {
  cart: CartItem[];
  addToCart: (item: Omit) => void;
  removeFromCart: (id: number) => void;
  updateQuantity: (id: number, quantity: number) => void;
  clearCart: () => void;
  getTotalPrice: () => number;
}

const CartContext = createContext(undefined);

export function CartProvider({ children }: { children: ReactNode }) {
  const [cart, setCart] = useState([]);

  const addToCart = (item: Omit) => {
    setCart((prevCart) => {
      const existingItem = prevCart.find((cartItem) => cartItem.id === item.id);
      if (existingItem) {
        return prevCart.map((cartItem) =>
          cartItem.id === item.id
            ? { ...cartItem, quantity: cartItem.quantity + 1 }
            : cartItem
        );
      }
      return [...prevCart, { ...item, quantity: 1 }];
    });
  };

  const removeFromCart = (id: number) => {
    setCart((prevCart) => prevCart.filter((item) => item.id !== id));
  };

  const updateQuantity = (id: number, quantity: number) => {
    setCart((prevCart) =>
      prevCart.map((cartItem) =>
        cartItem.id === id ? { ...cartItem, quantity: quantity } : cartItem
      ).filter(item => item.quantity > 0) // Eliminar si la cantidad llega a 0
    );
  };

  const clearCart = () => {
    setCart([]);
  };

  const getTotalPrice = () => {
    return cart.reduce((total, item) => total + item.price * item.quantity, 0);
  };

  return (
    
      {children}
    
  );
}

export function useCart() {
  const context = useContext(CartContext);
  if (context === undefined) {
    throw new Error('useCart must be used within a CartProvider');
  }
  return context;
}
