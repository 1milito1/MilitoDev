Integrando Firebase en tu app Android

Aprende paso a paso cómo conectar tu aplicación Android con Firebase para aprovechar sus servicios.

10 min lectura
Actualizado: 28/05/2025

Paso 1: Crear un proyecto en Firebase

Ve a Firebase Console y crea un nuevo proyecto. Sigue los pasos del asistente.

Paso 2: Conecta tu app Android a Firebase

  • Registra tu app: Añade el nombre del paquete y descarga el archivo google-services.json.
  • Coloca el archivo: Pon google-services.json en el directorio app/ de tu proyecto.
  • Gradle: Asegúrate de tener los plugins y dependencias necesarias:
// build.gradle (nivel raíz) classpath 'com.google.gms:google-services:4.3.15'
// build.gradle (nivel app) apply plugin: 'com.google.gms.google-services' implementation platform('com.google.firebase:firebase-bom:32.1.1') implementation 'com.google.firebase:firebase-auth' implementation 'com.google.firebase:firebase-database'

Paso 3: Configura Firebase Authentication

Habilita un proveedor como “Correo electrónico y contraseña” en Firebase Console (sección Authentication).

FirebaseAuth auth = FirebaseAuth.getInstance();

auth.createUserWithEmailAndPassword(email, password)
    .addOnCompleteListener(task -> {
        if (task.isSuccessful()) {
            FirebaseUser user = auth.getCurrentUser();
        } else {
            // manejar error
        }
    });

Paso 4: Conecta con Firebase Realtime Database

Puedes guardar y leer datos en tiempo real con el siguiente código:

DatabaseReference db = FirebaseDatabase.getInstance().getReference();

db.child("usuarios").child(userId).setValue("Milito Dev");

db.child("usuarios").child(userId).addListenerForSingleValueEvent(new ValueEventListener() {
  @Override
  public void onDataChange(DataSnapshot snapshot) {
    String nombre = snapshot.getValue(String.class);
  }

  @Override
  public void onCancelled(DatabaseError error) {
    // manejar error
  }
});

Conclusión

  • Creaste y vinculaste un proyecto Firebase con Android
  • Implementaste autenticación de usuarios
  • Guardaste y consultaste datos en tiempo real
