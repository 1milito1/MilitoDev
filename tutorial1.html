Creando tu primera app con Flutter

Guía completa para comenzar con Flutter y construir una aplicación multiplataforma desde cero.

15 min lectura
Actualizado: 13/05/2025

Introducción a Flutter

Flutter es el framework de Google para crear aplicaciones nativas compiladas para móvil, web y escritorio desde una única base de código. En este tutorial aprenderás:

  • Cómo instalar Flutter y configurar tu entorno de desarrollo
  • Crear y ejecutar tu primera aplicación Flutter
  • Entender la estructura básica de un proyecto Flutter
  • Modificar la aplicación predeterminada para crear algo personalizado

Requisitos previos: Conocimientos básicos de programación, tener instalado Android Studio o Xcode (según tu sistema operativo), y alrededor de 2GB de espacio libre en disco.

Paso 1: Instalación de Flutter

1

Descarga el SDK de Flutter

Visita el sitio oficial de Flutter y descarga la última versión estable del SDK para tu sistema operativo:

Descargar Flutter
2

Extrae el archivo descargado

Extrae el archivo zip descargado en una ubicación adecuada de tu sistema. Por ejemplo:

unzip ~/Downloads/flutter_macos_3.7.0-stable.zip -d ~/development

O si estás en Windows, puedes extraerlo usando el explorador de archivos a una ubicación como C:\src\flutter.

3

Configura el PATH

Agrega la carpeta flutter/bin a tu variable de entorno PATH. Esto te permitirá ejecutar el comando flutter desde cualquier ubicación.

Para Windows:

set PATH=%PATH%;C:\src\flutter\bin

Para macOS/Linux:

export PATH="$PATH:`pwd`/flutter/bin"

Para hacer este cambio permanente, agrega el comando a tu archivo .bashrc o .zshrc (macOS/Linux) o configura las variables de entorno del sistema en Windows.

4

Ejecuta flutter doctor

Ejecuta el siguiente comando para verificar que todo esté correctamente instalado y para identificar cualquier dependencia faltante:

flutter doctor

Este comando verificará tu entorno y mostrará un informe del estado de tu instalación de Flutter. Sigue las instrucciones para instalar cualquier componente faltante.

Paso 2: Configura tu editor

1

Elige un editor

Puedes usar Android Studio, IntelliJ IDEA o Visual Studio Code para desarrollar aplicaciones Flutter. Recomendamos VS Code por su ligereza y excelente soporte para Flutter.

Visual Studio Code - Editor recomendado para Flutter
2

Instala los plugins necesarios

Para VS Code, instala los siguientes plugins desde el marketplace:

  • Flutter - Soporte oficial para el framework Flutter
  • Dart - Soporte para el lenguaje de programación Dart
Instalación de los plugins de Flutter y Dart en VS Code

Paso 3: Crea tu primera aplicación

1

Genera un nuevo proyecto

Abre una terminal y ejecuta el siguiente comando para crear un nuevo proyecto Flutter:

flutter create mi_primer_app

Esto creará un directorio llamado mi_primer_app con todos los archivos necesarios para una aplicación Flutter básica.

2

Navega al directorio del proyecto

cd mi_primer_app
3

Ejecuta la aplicación

Conecta un dispositivo físico o inicia un emulador, luego ejecuta:

flutter run
Aplicación Flutter por defecto ejecutándose en un dispositivo

Si todo está configurado correctamente, deberías ver la aplicación de contador predeterminada en tu dispositivo o emulador.

Paso 4: Entiende la estructura del proyecto

1

Archivos y directorios principales

Un proyecto Flutter típico tiene la siguiente estructura:

mi_primer_app/
├── android/          # Archivos específicos de Android
├── ios/              # Archivos específicos de iOS
├── lib/              # Código Dart de tu aplicación
│   └── main.dart     # Punto de entrada de tu aplicación
├── test/             # Pruebas unitarias
├── pubspec.yaml      # Archivo de configuración del proyecto
└── README.md         # Documentación básica
2

El archivo main.dart

Abre el archivo lib/main.dart en tu editor. Este es el punto de entrada de tu aplicación y contiene el código de la aplicación de contador predeterminada.

import 'package:flutter/material.dart';

void main() {
  runApp(const MyApp());
}

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Flutter Demo',
      theme: ThemeData(
        primarySwatch: Colors.blue,
      ),
      home: const MyHomePage(title: 'Flutter Demo Home Page'),
    );
  }
}

class MyHomePage extends StatefulWidget {
  const MyHomePage({super.key, required this.title});

  final String title;

  @override
  State<MyHomePage> createState() => _MyHomePageState();
}

class _MyHomePageState extends State<MyHomePage> {
  int _counter = 0;

  void _incrementCounter() {
    setState(() {
      _counter++;
    });
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text(widget.title),
      ),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: <Widget>[
            const Text(
              'You have pushed the button this many times:',
            ),
            Text(
              '$_counter',
              style: Theme.of(context).textTheme.headline4,
            ),
          ],
        ),
      ),
      floatingActionButton: FloatingActionButton(
        onPressed: _incrementCounter,
        tooltip: 'Increment',
        child: const Icon(Icons.add),
      ),
    );
  }
}
3

Partes clave del código

  • main(): El punto de entrada de la aplicación Dart. Llama a runApp() con una instancia de tu widget principal.
  • MyApp: Un widget sin estado (StatelessWidget) que configura la aplicación básica con un tema y una página de inicio.
  • MyHomePage: Un widget con estado (StatefulWidget) que mantiene y actualiza el contador.
  • Scaffold: Proporciona una estructura básica de diseño de Material Design (AppBar, Body, FloatingActionButton).
  • setState(): Notifica al framework que el estado ha cambiado y que la interfaz debe reconstruirse.

Paso 5: Personaliza tu aplicación

1

Cambia el título y el tema

Modifica el widget MyApp para cambiar el título y el color principal:

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Mi Primera App',
      theme: ThemeData(
        primarySwatch: Colors.deepPurple,
      ),
      home: const MyHomePage(title: 'Bienvenido a mi App'),
    );
  }
}
2

Modifica la página principal

Cambia el widget MyHomePage para mostrar un mensaje diferente:

class _MyHomePageState extends State<MyHomePage> {
  int _counter = 0;

  void _incrementCounter() {
    setState(() {
      _counter++;
    });
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text(widget.title),
      ),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: <Widget>[
            const Text(
              '¡Hola Mundo Flutter!',
              style: TextStyle(fontSize: 24),
            ),
            const SizedBox(height: 20),
            const Text(
              'Número de veces que has presionado:',
            ),
            Text(
              '$_counter',
              style: Theme.of(context).textTheme.headline4,
            ),
          ],
        ),
      ),
      floatingActionButton: FloatingActionButton(
        onPressed: _incrementCounter,
        tooltip: 'Increment',
        child: const Icon(Icons.add),
      ),
    );
  }
}
3

Agrega un botón para decrementar

Modifica el método _incrementCounter y agrega un nuevo botón:

void _incrementCounter() {
  setState(() {
    _counter++;
  });
}

void _decrementCounter() {
  setState(() {
    if (_counter > 0) {
      _counter--;
    }
  });
}

@override
Widget build(BuildContext context) {
  return Scaffold(
    appBar: AppBar(
      title: Text(widget.title),
    ),
    body: Center(
      child: Column(
        mainAxisAlignment: MainAxisAlignment.center,
        children: <Widget>[
          const Text(
            '¡Hola Mundo Flutter!',
            style: TextStyle(fontSize: 24),
          ),
          const SizedBox(height: 20),
          const Text(
            'Número de veces que has presionado:',
          ),
          Text(
            '$_counter',
            style: Theme.of(context).textTheme.headline4,
          ),
        ],
      ),
    ),
    floatingActionButton: Column(
      mainAxisAlignment: MainAxisAlignment.end,
      children: [
        FloatingActionButton(
          onPressed: _incrementCounter,
          tooltip: 'Increment',
          child: const Icon(Icons.add),
        ),
        const SizedBox(height: 10),
        FloatingActionButton(
          onPressed: _decrementCounter,
          tooltip: 'Decrement',
          child: const Icon(Icons.remove),
        ),
      ],
    ),
  );
}
Aplicación Flutter personalizada con dos botones
